Sobre o Projeto

Este projeto investiga a popularidade de personagens da Marvel e da DC Comics no Brasil, com base em dados de 2024. O objetivo é responder três perguntas principais:

  • Qual é o herói mais popular no Brasil?
  • Qual é o vilão mais popular no Brasil?
  • Entre Marvel e DC, qual editora é mais popular no país?
